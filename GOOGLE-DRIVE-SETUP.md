## Step-by-Step Setup

### Step 1 — Create the folders
1. Go to drive.google.com
2. Create a folder called **NMGC Website**
3. Inside it, create one folder per league plus Club Documents

### Step 2 — Upload the PDFs
Upload each PDF into the correct folder using the exact filenames above.
Consistent naming means commissioners always know which file to replace.

### Step 3 — Get shareable links
For EACH file:
1. Right-click the file → **Share**
2. Change access to **"Anyone with the link"** → **Viewer**
3. Click **"Copy link"**
4. Paste the link into the website (or send to AJ to wire in)

### Step 4 — Convert the link format
Google Drive shareable links look like this:
```
https://drive.google.com/file/d/FILE_ID/view?usp=sharing
```

For the website, convert to direct download format:
```
https://drive.google.com/uc?export=download&id=FILE_ID
```

Just copy the FILE_ID from the middle of the link.

## How Commissioners Update Files

When standings, brackets, or any PDF needs updating:

1. Open Google Drive
2. Navigate to their league folder (e.g. **NMGC Website → Best Ball**)
3. Right-click the file to replace (e.g. **standings.pdf**)
4. Click **Manage versions** → **Upload new version**
5. Select the new PDF from their computer
6. Click **Done**

**The website link stays the same. Members see the new file instantly.**

No website login. No contacting AJ. No technical knowledge needed.

## When a Brand New Document Is Added

If a commissioner needs to add a file that has never existed before
(e.g. a new bracket mid-season):

1. Commissioner uploads the PDF to their Drive folder
2. Gets the shareable link and sends it to AJ
3. AJ pastes the link into the website HTML (2 minutes)
4. Done

This only happens for truly new documents — not updates to existing ones.
